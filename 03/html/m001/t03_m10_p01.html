Purpose

The purpose of the N2 input has the following effects:

  1. Transmittes rotational movement to the flyweights in the MFC governing system.
  2. Provides the N2 signal to the N2 servo governer.
  3. Provides N2 signal to the accel/decel computer system.
  4. Provides N2 to the maximum corrected speed limiting system.
