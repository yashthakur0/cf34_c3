Identification
  • The N2 control system, or hydromechanical core engine governing system, is part of the engine control system.
  • The N2 control system consists of:
    • Fuel metering system
    • Main fuel control governing
    • Acceleration and Deceleration computer system
    • Idle speed system
    • Altitude idle reset system
  • The primary component that controls N2 within the system is the Main Fuel Control which also positions the VG system based on input parameters.
