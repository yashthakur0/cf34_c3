Purpose

The T2C input has three primary functions:

  1. Provides feedback to the accel/decel computer system in the MFC to precisely regulate the fuel flow so that proper combustion takes place regardless of air temperature.
  2. Provides feedback to the accel/decel computer to operate the variable geometry system.
  3. Provides feedback to the idle speed system in the MFC to regulate the idle speed as a function of temperature.
