Purpose
  • The N2 control provides:
    • N2 speed governing variable geometry positioning
    • Idle speed control
    • N2 overspeed and over pressure protection.
  • Various sensors are used in the N2 control system and are inputs to the MFC effecting different aspects of the control systems. The sensors are:
    1. T2C; Idle speed system, Accel/Decel computer system,
    2. N2; MFC governing system, Idle speed system, Accel/decel computer system
    3. P3.0; Accel/Decel computer system
    4. P0; Altitude Idle Reset system
    5. VG system feedback; VG system operation.
