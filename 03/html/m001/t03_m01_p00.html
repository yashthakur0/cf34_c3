Identification

  • The engine control system is designed to provide the engine with the proper amount of fuel for combustion in all areas of operation within the aircraft flight envelope.
  • The system also includes automatic subsystems to protect the engine against stall and prevent the engine from exceeding it's mechanical limits.
  • Engine control is maintained through two sub-systems:
    • A hydro mechanical core engine speed governing system (N2)
    • An electrical fan speed governing system (N1)

  • The control system consists of the following components and sensors;
    • Main Fuel Control (MFC)
    • N1 Amplifier
    • Variable Geometry (VG)
    • N1 sensor
    • Alternator
    • T2 sensor
    • T2C sensor
    • P3.0 pressure
  • The control system interfaces with the flight deck through the Power Lever Angle (PLA), and an N1 speed switch.
