Purpose

The engine identification data plate is used to record the following data about the engine:

    • Assembly identification number
    • Model number
    • Manufacturing serial number
    • Type certification number
    • Production certification number
    • Thrust ratings (Max. Takeoff, Normal Takeoff, Max. Continuous)
    • Date of manufacture
    • Status of compliance with the Exhaust Emission Standards
