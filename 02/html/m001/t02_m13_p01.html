Purpose

The core cowl serves the following purpose:

    • It provides smooth aerodynamic surface for airflow aft of the fan cowling.
    • It provides outlet for the exhaust gases escaping from the engine.
    • It has a fire containment firewall that prevents the spread of fire between the engine and the airframe.
    • It provides internal passages for the flow of air for core and tail pipe cooling.
    • It provides openings for the engine drains.
    • It provides space for the Integrated Drive Generator (IDG) oil cooler.
    • It provides space for installation of components of the thrust reverser system.
