Operation

  • When the START switch is pressed and held for two seconds, the contacts in the switch close and the electric current flows from the battery to the start relay through the speed cutout switch.
    • The start relay is energized and energizes the solenoid valve assembly in the air starter valve.
    • This action closes the contacts in the position indicating switch assembly in the air starter valve and sends a signal to the EICAS system.
    • The white-colored ENGINE START message for the relevant engine is now displayed on the EICAS Secondary page.
  • The start relay also energizes the "A" or "B" ignition relay.
    • This action displays the amber-colored IGNITION message on the EICAS Secondary page.
  • During and after the start cycle, the EICAS Primary page displays the actual N2 and ITT for each of the engines.
  • When the engine N2 reaches approximately 55%, the power to the start relay is removed.
    • This deenergizes the solenoid valve assembly in the air starter valve.
    • This action turns off the white-colored ENGINE START message on the EICAS Secondary page.
