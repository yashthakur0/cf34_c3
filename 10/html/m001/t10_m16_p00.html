Identification

Engine control is maintained through two sub-systems: a hydro mechanical core engine speed governing system (N2) and an electrical fan speed governing system (N1). The control system consists of the following components and sensors; Main Fuel Control (MFC), N1 Amplifier, Variable Geometry (VG) vane system, N1 sensor, Alternator, T2 sensor, T2C sensor, P3.0 pressure. The control system interfaces with the flight deck through the Power Lever Angle (PLA), and an N1 speed switch. A failure of any one component can lead to a symptom which will indicate to maintenance the possible cause of the symptom. This could be a indication or an engine control issue.
